# Vending Machine Simulation

Console-based C++ application that simulates the behavior of a vending machine.

## Features

- product management
- purchase simulation
- change calculation based on the coins available in the machine
- data validation and exception handling

## Technologies

- C++
- STL
- Object-Oriented Programming (OOP)
- CMake

## Architecture

The application follows a layered architecture:

- Domain
- Repository
- Service
- UI

## How it works

The system simulates the purchase process.  
If a user inserts an amount **X** and selects a product with price **Y**,  
the application calculates the change (**X - Y**) using the coins available in the machine.
